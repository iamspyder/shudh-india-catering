# Shudh India - Design Direction (V2 Rich Premium)

This is the updated design direction after feedback: the website should feel like a **real premium catering brand site**, not a template or a sample.

## Final visual direction

- Dark luxury theme based on existing brand colors (`#2A0A0A`, gold accents).
- Strong hero storytelling + trust stats + rich sections.
- Premium cards, clear typography hierarchy, and high-quality motion.
- Conversion-focused flow from menu to checkout.

- `css/menu_design_preview.css` (new visual system and interactions)

## V2 key improvements

1. **Brand consistency**
   - Uses original Shudh India dark + gold brand mood.
   - Premium nav, hero, and footer aligned to existing identity.

2. **Real website structure**
   - Hero with clear value proposition and actions.
   - KPI strip.
   - Signature menu section with interactive tabs.
   - Process/timeline section.
   - Checkout concept block.
   - Testimonials and final footer.

3. **Richer interactions**
   - Reveal animations on scroll.
   - Card elevation + image hover zoom.

4. **Modern UX upgrades**
   - Better visual hierarchy for content blocks.
   - Cleaner CTAs.
   - Clear order process communication.
