# Shudh India — Premium Catering Platform
**Built by Ryvian Tech**

## Project Structure
```
shudh_india/
├── website/
│   ├── index.html        ← Homepage
│   ├── packages.html     ← Packages (Silver / Gold / Premium)
│   ├── inquiry.html      ← Custom Inquiry Form
│   ├── gallery.html      ← Photo Gallery
│   ├── videos.html       ← Videos Page
│   └── careers.html      ← Career Page + Job Listings
│
└── admin/
    ├── login.html        ← Admin Login (password: admin123)
    ├── dashboard.html    ← Overview + KPIs + Recent Leads
    ├── packages.html     ← Package Management
    ├── leads.html        ← Lead & Inquiry Management
    ├── gallery.html      ← Gallery Management
    └── settings.html     ← Content Control & Configuration
```

## How to Run
Open `website/index.html` in any browser. No server required.

## Admin Access
Navigate to `admin/login.html`  
Demo credentials: any email + password `admin123`

## Design System
- Primary: Deep Maroon `#2A0A0A`
- Accent: Royal Gold `#D4AF37` / `#E9C349`
- Background: Dark `#171212`
- Typography: Noto Serif (headlines) + Manrope (body)
- Style: Glassmorphism, Luxury Dark, Mobile-first
